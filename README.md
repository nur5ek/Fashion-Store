# Fashion-Store
frontend Fashion Store project 

Каждый перед работой делает:

git pull origin main


Потом редактирует свои файлы.

После — сохраняет и делает:

git add .
git commit -m "Сделала раздел Новинки"
git push origin main


Главное правило: всегда сначала git pull, потом работать и git push – тогда меньше конфликтов.
